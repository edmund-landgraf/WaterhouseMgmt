# Engagement Framework

This framework defines how Waterhouse should structure service engagements across management, infill, paving, and brokerage work.

## Property Management Engagements

Key clauses:

- Scope of managed communities or properties.
- Authority for maintenance approvals.
- Reporting cadence.
- Owner communication expectations.
- Accounting handoff responsibilities.
- Termination and transition process.

Operational attachments:

- Community roster.
- Emergency contacts.
- Vendor approval matrix.
- Reporting sample.

## Manufactured Home Dealership and Infill Engagements

Key clauses:

- Lot readiness responsibilities.
- Removal requirements.
- Utility constraints.
- Manufacturer and model selection.
- Delivery coordination.
- Setup inspection and occupancy handoff.
- Sales owner or project owner.

## Asphalt Paving Engagements

Key clauses:

- Scope of work.
- Surface preparation assumptions.
- Traffic and access plan.
- Weather delay handling.
- Change order process.
- Payment schedule.
- Closeout documentation.

## Brokerage Engagements

Key clauses:

- Property type and listing scope.
- Commission and compensation.
- Disclosure obligations.
- Marketing plan.
- Offer review and negotiation process.
- Closing coordination.

## Digital Workflow

1. Lead or inquiry is created.
2. Manager chooses engagement type.
3. System selects the matching template.
4. Required attachments are generated.
5. E-signature package is sent.
6. Signed documents become visible in the proper portal.
