# Operations SOP

This SOP defines repeatable standards for Waterhouse operations across service lines.

## Intake

For every inquiry:

- Identify contact type.
- Identify service line.
- Capture property or community.
- Capture urgency.
- Assign owner.
- Create next task.

## Property Management

- Log maintenance requests with community, lot/unit, resident, priority, and description.
- Track owner-facing issues separately from resident-facing updates.
- Keep documentation for inspections, notices, and vendor work.
- Use consistent status labels.

## Manufactured Home Infill

- Confirm lot readiness.
- Document removal needs.
- Confirm utility constraints.
- Track manufacturer, delivery date, setup date, and sales owner.
- Keep photo and inspection records.

## Asphalt Paving

- Capture site photos before estimate.
- Define scope category.
- Note access constraints and resident/community disruption.
- Track scheduled start/end and closeout.
- Store final scope and invoice documents.

## Brokerage

- Confirm property type.
- Confirm buyer/seller contacts.
- Track deal status and estimated value.
- Store disclosures and transaction documents.
- Keep communication history.

## Documentation

Every material workflow should produce:

- Source record.
- Assigned task.
- Owner/resident/vendor contact where applicable.
- Status.
- Supporting document or note.
- Audit event for major changes.
