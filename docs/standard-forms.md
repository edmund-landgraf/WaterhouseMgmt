# Standard Forms and Resources

This document tracks common forms and document categories that Waterhouse may need to standardize.

## Property Management Forms

- Resident intake form.
- Maintenance request form.
- Owner onboarding checklist.
- Community onboarding checklist.
- Vendor onboarding packet.
- Inspection checklist.
- Notice templates.

## Manufactured Home Sales and Infill Forms

- Buyer intake.
- Lot readiness checklist.
- Home selection worksheet.
- Delivery coordination checklist.
- Setup milestone checklist.
- Walkthrough checklist.

## Asphalt Paving Forms

- Site assessment form.
- Scope of work template.
- Estimate template.
- Change order form.
- Closeout checklist.
- Photo log template.

## Brokerage Forms

- Seller intake.
- Buyer intake.
- Property information sheet.
- Listing checklist.
- Transaction document checklist.

## Storage Guidance

Future portal storage should associate documents by:

- `ownerType`
- `ownerId`
- `category`
- `createdAt`
- `signedStatus`
