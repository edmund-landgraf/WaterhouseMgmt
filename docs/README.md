- [saas-component-map.md](./saas-component-map.md) - Software modules needed to support each operating function, with current provider examples.
- [app-architecture.md](./app-architecture.md) - Proposed Node, React, Tailwind, shadcn, and Express application structure.
- [product-roadmap.md](./product-roadmap.md) - Phased plan for turning the operating map into a production web app.
- [data-model.md](./data-model.md) - Initial entities and relationships implied by the scrape.
